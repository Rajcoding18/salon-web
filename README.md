# Glow & Grace — Salon Website

Attractive and responsive salon website built with plain HTML, CSS, and JavaScript.
Includes pages: Home, Services, Pricing, Contact.

## How to run locally
1. Extract the ZIP.
2. Open `index.html` in your browser.
3. Replace images in `assets/` (logo.png, hero.jpg) with your own photographs for best results.
